# 五音不全 — AI 智能音乐教学 App · 项目总结

## 项目概述

**「五音不全」** 是一款专为音准困难人群设计的 AI 智能音乐教学应用。用户无需任何乐理基础，通过闯关式训练 + 实时音高反馈 + AI 导师指导，逐步提升音准感知和演唱能力。

---

## 主题特色

### 🎨 奶油治愈系视觉风格

整体采用温暖柔和的奶油色系（`#FDF8F3` 暖白底 + `#E8B4A0` 陶土粉主色），搭配圆润大圆角（`1rem`）和毛玻璃导航栏，营造出低压迫感、友好亲切的学习氛围，降低音乐初学者的心理门槛。

三大乐理模块各有专属配色：
- **五线谱** — 草木绿 `#A8CBB7`
- **简谱** — 暖橙色 `#F2CFA4`
- **工尺谱** — 大地色 `#C8B8A8`

### 🧠 AI 贯穿全流程

- **AI 导师**：对话式智能问答，支持文字和语音输入，预设问题建议卡片，提供个性化音乐指导
- **AI 音色整曲学习**：上传清唱素材 → 搜索目标歌曲 → Seed-VC 音色替换 → 生成「我的声音版 AI 领唱」→ 整歌伴唱练习（含独立工作台：清唱素材面板、歌曲搜索面板、音色转换面板、伴唱控制面板）
- **K 歌房**：精选热门歌曲，滚动歌词 + 音高条，跟唱练习

## 核心功能模块

### 1. 新用户引导流程

| 步骤 | 内容 |
|------|------|
| 启动页 | 品牌展示 + 登录注册 |
| 问卷（6题） | 年龄、性别、音乐目标、唱歌痛点、偶像风格、音域认知 |
| 音域测试 | 实时音高检测，从最低音唱到最高音，自动分析声部类型 |
| 进入主页 | 所有数据持久化，支持后续重测音域 |

### 2. 教程闯关（核心训练模块）

三个训练模块，每个分 L1（基础级）/ L2（进阶级）/ L3（大师级）三个等级，支持自动升降级：

| 模块 | 题型 | 等级机制 |
|------|------|----------|
| 🎯 **音高听辩** | T1 双音比较 / T2 三音比较 | 80% 正确率晋级，55% 降级 |
| 🎵 **音程听写** | T3 音程比较 / T4 多音程配对 / T5 旋律听写 | 82% 晋级，55% 降级 |
| 🎤 **听音跟唱** | T6 单音模唱 / T7 短句跟唱 | 85% 晋级/掌握，58% 降级 |

**亮点机制**：
- 5 颗心的生命值系统 + Combo 连击加分
- 晋级/掌握时自动触发 **撒花粒子动画**
- 慢速播放辅助（用于音程比较，但会影响晋级评分）
- 答题后显示音名和唱名标注，帮助用户建立视觉-听觉关联
- 20 题/轮，中途退出不保存进度

### 3. 乐理闯关（知识体系）

三大门类合计 **17 个关卡**、**170 道练习题**，覆盖从入门到进阶的完整乐理知识链：

| 门类 | 关卡数 | 内容 |
|------|--------|------|
| 🎼 **五线谱**（staff） | 8 关 | 线间认知 → 中央C → 音名唱名 → 钢琴对应 → 全音半音 → 升降号 → 节奏 → 综合识谱 |
| 🔢 **简谱**（jianpu） | 5 关 | 数字音符 → 高低音点 → 节奏休止 → 音阶 → 升降号 |
| 📜 **工尺谱**（gongche） | 4 关 | 谱字认知 → 简谱对应 → 旋律转换 → 文化挑战 |

**闯关机制**：
- 每关 10 题，≥80% 三星，≥50% 两星，>0% 一星
- 「上一关必须 ≥2 星」解锁下一关
- 工尺谱需简谱第 2 关解锁
- 模块集满 85% 星星解锁金色边框 + 庆祝动画
- 错题本功能（自动收录错题，支持标记已掌握）
- 复习模式（随机抽取历史错题）

### 4. 个人主页

| 功能 | 内容 |
|------|------|
| 快速统计 | 连续天数、今日练习数、进行中模块数 |
| SingTI 音乐人格 | 唱《小星星》+ 16 道题解锁专属音乐人格，支持分享 |
| 音域展示 | 可视化音域条 + 声部分类（女高音/女中音/男高音等） |
| K 歌房 | 精选歌曲跟唱 |
| 自我评估 | 综合音乐能力评估 |
| 今日练习统计 | 动态统计时长、题数、准确率 |
| 成就系统 | 7 个成就徽章（初试啼声、进阶级、大师、持之以恒等） |
| 学习进度 | 全模块进度一览 |

### 5. 节奏训练（独立模块）

6 个单元，每单元 1-4 关，支持分步渐进式节奏训练。

## 技术架构

### 前端

- **框架**：Next.js 16 + React 19 + TypeScript（Turbopack 构建）
- **UI 组件**：shadcn/ui + Radix UI 原语 + Tailwind CSS v4
- **动画**：Framer Motion（页面过渡、撒花粒子、进度环动画）
- **音高检测**：Web Audio API（OscillatorNode 播放 + 实时音高分析）
- **数据持久化**：localStorage 进度引擎（乐理进度、教程等级、错题本）
- **状态管理**：React Context + useReducer

### 后端

- **框架**：FastAPI + SQLAlchemy + PostgreSQL
- **AI 服务**：Seed-VC 音色替换 / AI 聊天接口
- **API 风格**：RESTful，统一响应格式

### 亮点技术

- 30+ Radix UI 原语组件完整集成
- 自定义 `useRef` 防抖 + localStorage 缓存层实现离线可用
- 实时音高检测与音准评分算法
- 多题型渲染引擎（T1-T7 七种交互模式）
